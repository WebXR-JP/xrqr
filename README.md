# XRQR

Quest のテキスト入力困難問題を解決するWebアプリケーション。PC/スマホで生成したQRコードをQuest のパススルーカメラで読み取り、クリップボードに同期します。

**アプリケーションURL**: https://xrqr.net

## 開発

```bash
# セットアップ
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## デプロイ

main ブランチにpushすると自動的に Cloudflare Pages にデプロイされます。
